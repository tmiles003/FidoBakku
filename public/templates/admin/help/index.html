Evaluations

Evaluations are contained in loops, each loop representing a period of evaluation.
An evaluation is for 1 person, user evaluations are created for each evaluation. The form used is the one set in the Forms section for that user. An evaluation can be in one of 4 modes:

  • "New" - only Managers & Admins have visibility, while they set up all the user evaluations
  • "Evaluation" - it's available for the users to complete the user evaluations
  • "Feedback" - it's available to Managers and Admins to do the evaluation feedback
  • "Done" - the user can see the completed evaluation

Evaluation mode can be toggled between Evaluation & Feedback on the assignment page. The evaluation is marked complete from the evaluation itself.

Forms

Forms are your templates of competencies for rating. Add sections and competencies as you see fit, you can move them up and down, they will keep their order.

Shared forms

To avoid unnecessary repetition, you can create shared forms. They are re-usable form components that you can prefix to other forms, so for example you could have company-wide competencies, prefixed to team-wide competencies, prefixed to role competencies.

Assigned users

Users are assigned to a form, it is the form that will be used for their evaluation.
Only users with an assigned form will be available in the drop-down when you create evaluations.

Staff

Users are created with a random password. The first time they sign in to the service, they will need to use the "Reset password" functionality to get started.

There are 3 access levels:

  • Employee: can view their own goals and evaluations, and any assigned user evaluations in "Evaluation" mode
  • Manager: as Employee, + can manage Forms, Loops & Evaluations (including User Evaluations), as well as Feedbacks
  • Admin: as Manager, + can manage Staff, & Account

Users can be assigned to a team. The users at the top of the page are not assigned to any team.

Avatars

To personalise their profile, each user can create a Gravatar. Their chosen picture will then appear instead of the FidoBakku logo.

Teams

Deleting a team does not delete the users in that team.

Account

An account must have an admin user as owner.

Deleting data

Deleting data is an immediate, permanent, unreversible action, after the confirmation prompt.
